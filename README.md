PaperClip is an AI model which can go through a research and predict its publishability
it will verify whether a paper is publishable or not
if publishable -
     it determines which conference is best for the paper to host
if not publishable it show the reason for it
it gives a rationale for both publishable and non publishable papers
it also detects plagiarism in the papers
i have trained the model with a limited data set of 15 papers
with more number of datasets, we can make its scores improve by a recognizable amount
